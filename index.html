혼자 하기엔 너무 어렵죠

시계 마니아라면 한 번쯤 겪어봤을 불편함들이에요.

🔍
원하는 교환 상대 찾기가 너무 힘들어요
내 시계를 원하면서, 내가 원하는 시계를 가진 사람을 카페·오픈채팅에서 혼자 찾기란 하늘의 별 따기예요.
😟
가품이 무섭고, 사기도 걱정돼요
수백만 원짜리 시계 교환인데 상대방을 믿는 수밖에 없는 현실. 검증할 방법이 없어요.
⌚
여러 개를 정리해서 드림워치로 가고 싶어요
안 차는 시계 두세 개를 정리해 드림워치 하나로 가고 싶은데, 이 과정을 한 곳에서 해결할 수가 없어요.
CRCL이 하는 일

교환을 다시 설계했어요

시계 교환의 세 가지 문제를 한 번에 해결해요.

🔄
자동 교환 매칭
원하는 시계 우선순위만 입력하면 1:1 또는 다자간으로 자동 탐색해요.
🔍
정품 검수 보장
CRCL이 물건을 직접 수령해 확인 후 발송해요. 가품 걱정 없어요.
🎯
드림워치 여정
목표 시계를 설정하면 보유 시계를 팔 때마다 진행률을 보여줘요. 여러 개를 정리해 드림워치 하나로 가는 여정을 함께해요.
이렇게 작동해요

단 세 단계로

복잡한 건 CRCL이 알아서 해요.

1
내 시계 등록 + 원하는 시계 입력
브랜드·모델·상태·사진을 등록하고, 원하는 시계를 1~3순위로 설정해요. 드림워치가 있다면 목표로 설정하세요.
2
CRCL이 매칭을 찾아드려요
1:1이 안 되면 A→B→C→A 다자간 체인으로 자동 탐색해요. 매칭이 성사되면 알림이 와요.
3
CRCL이 검수하고, 드림워치가 도착해요
시계를 CRCL로 보내면 정품 확인 후 상대방에게 발송해요. 현금 없이, 사기 걱정 없이.
